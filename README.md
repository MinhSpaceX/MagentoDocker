# MagentoDocker

## Command to pull files:
```
curl -L -o MagentoDocker.zip https://github.com/MinhSpaceX/MagentoDocker/archive/refs/heads/main.zip
```

## After pull, extract and go into extracted directory, then run:
```
chmod -R +x command
command/start
command/download
command/install
command/start
```
